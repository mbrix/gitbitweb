Bitcoin bounties for Github projects

1. Add the webhook to Github

Under settings for your github repository set the following webhook: http://api.gitbit.io/webhook

2. Automatically create bounties

!gitbit-bounty [bountyname] Please do this thing.
Any commit messages you add can include bounties that will automatically create bitcoin receiving addresses and list them here.

3. Claim bounties

!gitbit-claim [bountyname] 1JQntWJbppSqzco54kEow5D8ZMygQmSiHa
Redeem a bounty in your commit message. Merged code with accepted bounties will pay them out to the specified address.
